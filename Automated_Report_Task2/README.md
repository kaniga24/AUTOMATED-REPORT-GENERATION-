# Automated Report Generation

This project is part of the CodTech Internship Task-2.

## Features
- Reads data from a CSV file
- Analyzes the data using Python
- Generates a formatted PDF report using ReportLab
- Simple frontend to upload CSV and generate report

## Technologies Used
- Python
- Flask
- Pandas
- ReportLab
- HTML

## Project Structure
Automated_Report_Task2/
├── backend/
│   ├── app.py
│   ├── data.csv
├── frontend/
│   ├── index.html
├── requirements.txt
└── README.md

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run backend:
   python backend/app.py
3. Open frontend/index.html in browser
